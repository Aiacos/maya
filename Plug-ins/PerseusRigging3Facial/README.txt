What’s new in Perseus Rigging 3.0    
---------------------------------------------------------------
New tweaker layer for the lips and eyebrows
New squash&stretch layer for muzzle and forehead
Pupil&Iris setup
New user interface
Customizable joint count for eyelid and lips
Jaw line controlles
Forehead controllers
Improve eyebrow Controllers(4 to 5)
Improve lip controllers (5 to 9)
update tongue rig (scalable)
Maintain Max influences Jnts
Added Space switcher for the Eye aim Controller
Export & import controllers
added world controllers
Soft Ik for legs
MG picker UI
Fixed twist joints issue(Ik mode)
Compatibility : 2017, 2018, 2019,2020
