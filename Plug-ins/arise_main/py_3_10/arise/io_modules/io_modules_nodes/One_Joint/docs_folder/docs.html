One Joint

Creates either a ctrl, a joint, or both.

Guides:

  • '01 Guide' - Position and orient where you want the ctrl and/or joint.

Attributes:

  1. 'Create' - Choose between creating just the ctrl, just the joint, or both.
  2. 'Ctrls Scale' - Scale the node ctrl. Note that the attachments 'Ctrls Settings' and 'CtrlsShape' will override this. [Video] 
  3. 'Clean Transformations' - If checked, the zeroed pose will be the same as the bind pose; if unchecked, when zeroing the ctrls, they will align with a world axis specified in the following two attributes.
  4. 'World Orientation' - The world axis the ctrls will align with when zeroed. Usually, this attribute's default value is the correct value.
  5. 'World Orient Twist' - Along with 'World Orientation', defines the ctrls zeroed pose. Usually, the default value of 0 is the correct value.
  6. 'Expose RotateOrder' - Exposes the ctrls 'RotateOrder' attribute in the Channel Box. [Video]
  7. 'Secondary Ctrls' - Secondary ctrl is added under the ctrl to help prevent gimbal lock.
    A visibility attribute is added to the parent ctrl. [Video]
  8. 'Skinning Joint' - Tagged skinning joints can be selected easily from the 'Helper Tools', and they are used by some attachments.

Connections:

  • 'Input' - Drives the node.
  • 'Output' - An output at either the joint or ctrl.
