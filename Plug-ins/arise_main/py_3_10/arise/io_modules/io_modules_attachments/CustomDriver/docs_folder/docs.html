CustomDriver

By using 'CustomDriver', any Maya transform can drive a connection.

Two Maya transforms are actually constrained by connection attributes (attributes with green circles). With 'CustomDriver', you can specify any Maya transform as a driver.

Attributes:

  • 'Method' - Dropdown with two options that enable one of the attributes below.
  • 'List' - All the ctrls and joints associated with the node to select one as the driver.
  • 'Node Name' - You can enter any Maya transform name here, even from a different node, though this might cause problems, so only transforms belonging to the node should be used.
    You can load a selected transform by using the arrow button. Check your selection by looking at the indicator '*' color and tooltip.
  • 'Custom Driver' - Only appear on the node shape. When a connection is drawn, the transform specified in the previous attributes will be used.

Notes:

  • There are different types of connections: matrix constraint, parent constraint, orient constraint, point constraint, parenting, and direct connection, but the type of connection is determined by the driven attribute (check out 'CustomDriven').
  • Note that if the transform gets renamed, you'll need to update this attribute with his new name.
