JointsSettings

'JointsSettings' exposes the settings of each of the node's joints.

Attributes:

  • 'skinning_jnt' - Select which joint is tagged as a skinning joint. Some attachments rely on tagged skinning joints. You can also select skinning joints using the 'Helper Tools'.
  • 'radius' - Adjust the joint size.
  • 'template' - Make the joint un-selectable.
  • 'label_side - Changes the joint 'side' attribute under 'Joint Labelling'. This can be useful when using Maya's 'Mirror Skin Weights' with the 'Label' option.
