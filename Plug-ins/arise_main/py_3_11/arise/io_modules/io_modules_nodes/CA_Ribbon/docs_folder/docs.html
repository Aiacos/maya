CA Ribbon

Cartoon ribbon is not a specific body part but can be used wherever its behavior is desired, for example, eyebrows.

Guides:

  • 'Ribbon Guides' - Position where you want the ribbon ctrls. The number of these guides is determined by the attribute 'Ctrls Count'.

Attributes:

  1. 'Ctrls Scale' - Scale all the node ctrls. Note that the attachments 'Ctrls Settings' and 'CtrlsShape' will override this. [Video]
  2. 'Clean Transformations' -  If checked, the zeroed pose will be the same as the bind pose; if unchecked, when zeroing the ctrls, they will align with a world axis specified in the following two attributes. [Video]
  3. 'World Orientation' - The world axis the ctrls will align with when zeroed. [Video]
  4. 'World Orient Twist' - Along with 'World Orientation', defines the ctrls zeroed pose. Usually, the default value of 0 is the correct value. [Video]
  5. 'Expose RotateOrder' - Exposes all the ctrls 'RotateOrder' attribute in the Channel Box. [Video]
  6. 'Secondary Ctrls' - Secondary ctrls are added under some ctrls to help prevent gimbal lock.
    A visibility attribute is added to the parent ctrl. [Video] 
  7. 'Volume' - Adds 'Auto Volume' and 'Manual Volume' attributes.
  8. 'Mid Joints Aim' - Mid driver joints will aim at each other; this means that mid ctrls rotation will only affect twist, giving the animator less control, but it will be easier to pose the ribbon to flow.
  9. 'Ctrls Count' - The number of ctrls/guides; modifying this attribute requires a 're-template'.
  10. 'Ribbon Joints' - The number of skinning joints along the ribbon. [Video]
  11. 'Ribbon Micro Ctrls' - Add a ctrl for every ribbon skinning joint. [Video]
  12. 'Bezier Ctrls' - Add ctrls that deform the ribbon like a Bezier curve. [Video] 
  13. 'Bezier Ctrls Offset' - Position offset of the Bezier ctrls from the joint chain.

Connections:

  • 'Root Input' - Drives the root of the ribbon.
  • 'Tip Input' - Drives the tip of the ribbon.
  • 'Root Output' - An output at the 'ribbon_0_driver' joint.
  • 'Tip Output' - An output at the last 'ribbon_#_driver' joint.
