AddJoints

The 'AddJoints' attachment allows you to add custom joints and ctrls to the node.

After setting the number of joints in the attachment settings, new guides will appear in Template mode, enabling you to position and orient the new joints and ctrls.

Attributes:

  • 'Enable Joints Color' - Toggle to enable coloring of the joints.
  • 'Joints Color' - Select the color for the joints.
  • 'Ctrls Color' - Select the color for the controls.
  • 'Joints Count' - Specify the number of custom joints to add.
  • 'Add Joints' -
    • 'joint_parent' - Specify the parent joint for this joint. Select "None" for no parent.
    • 'create_ctrl' - Enable or disable creating a ctrl to drive the joint.
    • 'ctrl_parent' - If a ctrl is created, you can specify another ctrl or joint to drive it.
  • 'Reset Changes' - Remove any changes to the above 'Add Joints' table.
  • 'Visibility Attribute' - Enable adding an attribute to control the visibility of the custom ctrls.
  • 'Add Attribute On' - Select the ctrl to which the visibility attribute would be added to.

Notes:

  • You can rename the newly created joints/ctrls using the 'Rename' attachment.
  • Modify individual joints and ctrls, such as changing ctrls shapes, using the 'JointsSettings' and 'CtrlsSettings' attachments.
