CtrlsShape

With 'CtrlsShape', you have the ability to manually modify the shape of ctrls.
However, make sure to save your modifications on the attachment.

Steps:

  1. Right-click hold on a ctrl in the Maya scene and choose 'Control Vertex'.
  2. Modify the ctrl CVs.
  3. When done, go to the attachment settings in the Arise Attribute Editor, and press 'Save Modifications'.

Notes:

  • You can only modify the position of existing CVs; you cannot add or remove CVs or shapes.
  • Please note that 'CtrlsShape' only saves modifications made to ctrls belonging to the node the attachment is on. If you make changes to ctrls on other nodes, they won't be saved.
  • After adding a 'CtrlsShape' to a node, changes to the parent node 'Ctrls Scale' attribute will be ignored.
  • After adding a 'CtrlsShape' to a node, changes to some of the attributes in 'CtrlsSettings' will be ignored.

As of version 1.01.00, a 'Scale Ctrls' slider is available in the 'Helper Tools' menu that allows you to scale selected ctrls or all the ctrls in the scene if none are selected. Remember to save your modifications after using this feature.

To further streamline the process, you can use the 'Batch' > 'Add CtrlsShape To Selected Nodes' option to add this attachment to multiple nodes at once. Then, use the 'Batch' > 'Save All CtrlsShape Attachments' option to save all your modifications with a single click. This can save you a lot of time and effort.
