AddGroupAbove

'AddGroupAbove' simply adds an empty group above specified ctrls.

For advanced riggers/TDs who may want to modify the rig but need a buffer group above the ctrls.
You can also use this technique with motion-capture, where you plug the motion-capture animation into the groups above the ctrls, leaving the ctrls untouched for manual animation fixes.

Attributes:

  • 'Groups Suffix' - The name suffix for groups. If left empty, will use 'buffer_grp'.
  • 'Add Groups' - Select which ctrl will get a group above it.
  • 'Reset Changes' - Removes any changes made to the 'Add Groups' table attribute.

Notes:

  • There can be more than one 'AddGroupAbove' attachment per node which stacks empty groups above ctrls.
