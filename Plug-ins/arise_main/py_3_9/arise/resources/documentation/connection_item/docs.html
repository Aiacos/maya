Connection

A Connection, also know as 'Connection Item', allows you to drive an attribute by another attribute.

Connections can only be made between two attributes on different nodes, two attributes of the same type (slots with the same color), and between an output slot and an input slot.
The most important connection type are between a driver and a driven attribute. Those connections drive the rig.

How to create a connection.

To create a connection: left click on a slot and drag to another node's slot. only highlighted slots are valid connections slots. A driven attribute will appear yellow in the attribute editor and it's value cannot be modified without first breaking the connection.
To modify a connection: click on an existing connection and drag one if it's ends to a different slot.

How to delete a connection.

To delete a connection: 1) Select the connection and press 'Delete' (Del) button.
2) Click and drag on the connection until the line turns to a dashed line then release on an empty spot. 3) Right click on a connection and from the context menu choose 'Delete Connection' 4) In the attribute editor right click on the connected attribute name and choose 'Break Incoming Connection'.

More information.

Selecting a connection will display information in the attribute editor about the connection such as data type, driver and driven attribute names, their values and minimum and maximum limits.
Note: if an attribute has an incoming connection value that is beyond his minimum and maximum range, it will clip it's incoming value.
Right clicking on the connection allows you to 'Delete Connection', 'Select Source Node' (connection driver node), 'Select Destination Node' (connection driven node).
