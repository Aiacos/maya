Parent

The 'Parent' attachment allows you to manually parent objects to the rig.

We could say, for example, that our character has a button on his shirt and that it doesn't need it to be skinned since it can be parented to the node's joint/ctrl/any transform. With the 'Parent' attachment, we can manually parent what we want under the node's rig parts. Then we can press the button 'Add Parent Tags' within the attachment settings. Now when switching to 'Template' mode the parented objects will be moved to the 'Dump Grp' but re-parented back when switching to 'Build' mode. 

Attributes:

  • 'Add Parent Tags' - Finds objects that are parented under the node's Maya objects and adds a 'Tag' to them so that they can be re-parented when needed.
  • 'Remove Parent tags' - Remove any parent tags associated with this node.
  • 'Print Parent Tags' - Prints the objects with the parent 'Tag' to the 'Log Viewer'.

Notes:

  • Remember that this attachment only tags objects parented to Maya nodes that belong to its parent Arise node. It may be necessary to add this attachment to more than one Arise node.
  • This setup will need to be redone if the parent node is renamed.
