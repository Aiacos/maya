CtrlsLimitAttributes

A 'CtrlsLimitAttributes' attachment does exactly what Maya's 'Limit Information' does.
Useful if you want to restrict the translation, rotation, and scaling values an animator can use when animating a ctrl.

Attributes:

  • 'Ctrls Limit Attributes' - Each ctrl has a column for enabling min value, min value, enabling max value, and max value for each of the ctrl transformation attributes.
  • 'Reset Changes' - Removes any modifications made to the 'ctrl Limit Attributes' table attribute.
