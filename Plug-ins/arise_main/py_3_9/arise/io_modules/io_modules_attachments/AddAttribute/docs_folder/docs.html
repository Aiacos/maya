AddAttribute

The 'AddAttribute' attachment is used to add a new attribute to a ctrl or joint. It has the same options as Maya's 'Add Attribute'.

Attributes:

  • 'Choose Transform' - Select a ctrl or joint to add an attribute to.
  • 'Attribute Name' - New attribute name.
  • 'Make Attribute'-
    • Keyable: Allows attribute to be keyed.
    • Displayable: The attribute will not be keyable but will appear in the Channel Box.
    • Hidden: The attribute will be hidden (and not keyable).
  • 'Data Type' - Check Maya's docs for more info about data types.
  • 'Default Value' - The default value for the attribute.
  • 'Minimum' - The minimum value for the attribute.
  • 'Maximum' - The maximum value for the attribute.
  • 'Enum Names' - If the data type is 'Enum', you can enter the names of enum options here.

Notes:

  • A node can have more than one attachment of this type.
  • Another attribute can be connected to this attribute using another 'AddAttribute' attachment or a 'ConnectAttribute' attachment.
