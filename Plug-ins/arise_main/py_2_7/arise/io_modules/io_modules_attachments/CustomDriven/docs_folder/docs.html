CustomDriven

By using 'CustomDriven', any Maya transform can be driven via a connection.

Two Maya transforms are actually constrained by connection attributes (attributes with green circles). With the 'CustomDrivern' attachment, you can specify a driven Maya transform as well as how it should be driven.

Attributes:

  • 'Method' - Dropdown with two options that enable one of the attributes below.
  • 'List' - All the ctrls and joints associated with the node to select one as the driven.
  • 'Node Name' - You can enter any Maya transform name here, even from a different node, though this might cause problems, so only transforms belonging to the node should be used. You can load a selected transform by using the arrow button. Check your selection by looking at the indicator '*' color and tooltip.
  • 'Custom Driven' - Also appears on the attachment shape. Will appear yellow once the attribute is connected.
    You can also input the name of a transform.
  • 'Connection Type' - Select the type of connection you want from a dropdown list. Depending on the selection, the attributes below will either be enabled or disabled.
  • 'Maintain Offset' - Maintains the current position/rotation of the driven transform.
  • 'Force Connection' - Disconnects any existing connections. When unchecked, any attribute already connected will be skipped.
  • 'Force When Locked' - When unchecked, locked attributes are skipped.
  • 'Attributes' - List of translation, rotation, and scale attributes. Unchecked attributes are skipped.
  • 'Aim Constraint Settings' - Same settings as Maya's aim constraint.
