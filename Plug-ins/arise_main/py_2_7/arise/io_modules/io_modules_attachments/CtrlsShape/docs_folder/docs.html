CtrlsShape

The 'CtrlsShape' attachment allows you to manually modify the shapes of the ctrls belonging to the node it’s placed on.

Alternative Methods:

You can also adjust ctrl size and shapes using:

  • The Ctrls Scale attribute (available on most nodes), or
  • The CtrlsSettings attachment.

Full Curve Data (New in v1.13.00):

As of version 1.13.00, CtrlsShape can store full NURBS curve data for each ctrl.
Enabling the new checkbox “Store Full Curve Data” lets you modify ctrl shapes beyond simply moving existing CVs, you can: replace or delete shapes, add or remove CVs, and create entirely new shapes.

Note: Enabling this option slightly increases build time, and .ions files will be a bit larger. 

 

Option A – Modify Existing CVs:

  1. In Maya, right-click and hold on a ctrl and select Control Vertex.
  2. Move the existing CVs to adjust the shape.
  3. In the Arise Attribute Editor, open the CtrlsShape attachment and press Save Modifications

With this option, you can only reposition existing CVs, you cannot add, remove, or replace shapes.

 

Option B – Full Shape Editing (v1.13.00+):
  1. Modify the ctrl shape freely: Replace shapes, delete CVs, or create new ones.
  2. In the CtrlsShape attachment settings, enable Store Full Curve Data.
  3. Press Save Modifications.

This option may slightly affect build time.

Important Notes:
  • Scope:
    CtrlsShape only saves modifications to ctrls belonging to the node it’s attached to.
    Changes made to ctrls on other nodes will not be saved.
  • Interaction with Other Features:
    • After adding a CtrlsShape, changes to the parent node’s Ctrls Scale attribute will be ignored.
    • Some attributes in CtrlsSettings may also be ignored after adding this attachment.
  • Helper Tools (since v1.01.00):
    A Scale Ctrls slider in the Helper Tools menu lets you scale selected ctrls or all ctrls in the scene if none are selected.
    Remember to save your modifications afterward.
  • After making any changes, make sure to save your modifications directly from the attachment.

 

Batch Workflow:

To streamline large projects:

  • Use Batch --> Add CtrlsShape To Selected Nodes to apply the attachment to multiple nodes at once.
  • Use Batch --> Save All CtrlsShape Attachments to save all ctrl modifications in one click.

These options can significantly reduce repetitive work when handling multiple nodes.
